.. _at_intro:

introduction
============
    
    The "at" character (``@``) is the character used in Genro to begin:
    
    * a :ref:`relation`
    * an :ref:`inverse_relation`
    * a :ref:`relation_path`
    
    Through these :ref:`relations` you can handle data between database tables.
