.. _readonly_def:

definition and description
==========================

    ::
    
        readOnly = BOOLEAN
        
    where BOOLEAN is ``True`` or ``False``.
    
    Boolean. If ``True``, the element that carries the readOnly attribute is
    in read-only mode.
    
.. _readonly_validity:

validity
========

    It works on every object.
