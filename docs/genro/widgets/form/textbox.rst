=========
 textbox
=========

.. currentmodule:: form

.. class:: textbox -  Genropy textbox

	A simple textbox.
	
		+-----------------------+---------------------------------------------------------+-------------+
		|   Attribute           |          Description                                    |   Default   |
		+=======================+=========================================================+=============+
		| ``default``           | Add a default text                                      |  ``None``   |
		+-----------------------+---------------------------------------------------------+-------------+
		| ``font_size``         | CSS attribute                                           |  ``1em``    |
		+-----------------------+---------------------------------------------------------+-------------+
		| ``text_align``        | CSS attribute                                           |  ``left``   |
		+-----------------------+---------------------------------------------------------+-------------+

		Example::

			pane.textbox(value='^box')
