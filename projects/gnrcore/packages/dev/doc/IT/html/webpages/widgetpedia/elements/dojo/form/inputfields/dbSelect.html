Field that does a real time db query on inserted value and returns a popup of matching values
 
Campo che compie rapide query in realtime e restituendo un set di record compatibili con la stringa inserita come valore di ricerca.
I record risultanti compatibili vengono mostrati in una piccola griglia a tendina sottostante. Una volta scelto il valore tra questi, viene inserito nel campo la pkey del record scelto. Questo widget è usato in genre per i campi i relazione.
